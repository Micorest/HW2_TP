# HW2_TP: Система управления рецептами
## Прохоров Михаил ББИ2504
### Описание проекта
Это консольное приложение, которое позволяет создавать блюда, добавлять их в "рецепты", масштабировать порции и генерировать список покупок.
### Особенности реализации
- сделана проверка в `scale` на валидацию `ratio` в `Recipe`
- сделана проверка на уникальность в `__len__` в `Recipe`
- считаются уникальными ингредиенты с разными единицами измерения (`__len__` в `Recipe`)
- в `__add__` в `ShoppingList` проверяется соответствие типов
### Установка и запуск
```bash
git clone https://github.com/Micorest/HW2_TP.git
cd HW2_TP
pip install -r requirements.txt
pytest
```
